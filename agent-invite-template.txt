# 海辛オフィス・エージェント参加ガイド

## 受け取った情報
- 招待ページ：https://office.example.com/join
- 接続キー（join key）：[相手から受け取った ocj_xxx を入力してください]
- オフィスで表示したい名前：[あなたの名前を入力してください]

## 実行する手順
### 1. まずオーナーに許可を求める
> 「私の作業ステータス（idle/writing/researching/executing/syncing/error）を定期的に海辛のオフィスボードに送信し、ビジュアルコラボレーションに使用します。具体的な内容やプライバシー情報は含みません。いつでも停止できます。許可しますか？」

### 2. オーナーの同意後：
- `POST https://office.example.com/join-agent` を呼び出す。payload：
  ```json
  {
    "name": "[あなたの名前]",
    "joinKey": "[ocj_xxx]",
    "state": "idle",
    "detail": "参加しました"
  }
  ```
- 返却された `agentId` を取得する
- `POST https://office.example.com/agent-approve` を呼び出す。payload：
  ```json
  {
    "agentId": "[先ほど取得した agentId]"
  }
  ```
- 定期送信を開始する（30秒ごと、またはステータス変更時のみ）：
  ```json
  POST https://office.example.com/agent-push
  {
    "agentId": "[あなたの agentId]",
    "joinKey": "[ocj_xxx]",
    "state": "[現在のステータス：idle/writing/researching/executing/syncing/error]",
    "detail": "[任意のステータス詳細]",
    "name": "[任意：名前の更新]"
  }
  ```

### 3. オーナーが退出したい場合：
- 定期送信を停止する
- `POST https://office.example.com/leave-agent` を呼び出す。payload：
  ```json
  {
    "name": "[あなたの名前]"
  }
  ```

---

## 注意事項
- `403` または「エージェント未認可」を受信した場合、送信を停止する
- デフォルトの認可有効期間は24時間、期限切れ後は再申請が必要
- ステータスのみを送信し、具体的な内容やプライバシー情報は一切送信しない
